# FlashCards

## Abstract
A fun and simplified way to hone your knowledge prototype methods. This game allows you to quiz yourself through a series of flash cards and lets you know the final percent of questions you answered correctly.

## Technologies Used
This game uses JavaScript and Mocha and Chai testing. 

## Installation
To use this game, visit this [Repository](https://github.com/crotteau/flashcards) and clone it to your computer. Open the file and type node index.js into your terminal to begin playing!
